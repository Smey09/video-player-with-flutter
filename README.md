## 4. Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Smey09/video-player-with-flutter.git

2. install package
   ```
   flutter pug get
   ```
3. Run app
   ```
   flutter run
   ```
4. if error version:
   1. update flutter version :
      ```
      flutter upgrade
      ```
   2. update package :
      ```
      flutter pug upgrade
      ```
